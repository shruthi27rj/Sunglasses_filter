# Sunglasses_filter
A real-time face filter application that detects faces using Haar cascades and overlays a sunglasses image using OpenCV.

## Overview
This project applies a sunglasses filter to a user's face in real time using OpenCV and Haar cascade classifiers.

## Features
- Real-time face detection
- Overlay of sunglasses image
- Uses webcam input

## Technologies
- Python
- OpenCV
